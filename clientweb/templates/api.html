Rest API Info

LobbyREST
  [GET] {{ url_for('lobbyrest_blueprint.rest_lobby_get_gametypes') }}     Get the list of types of games running on this platform
  [GET|POST] {{ url_for('lobbyrest_blueprint.rest_lobby_get_games') }}     Get the list of active games, Add to the list of active games
  [GET|DELETE|PATCH] {{ url_for('lobbyrest_blueprint.rest_lobby_get_game_info',gameid='gameid') }}     Get or interact with a specific game
  [GET|POST] {{ url_for('lobbyrest_blueprint.rest_lobby_get_players',gameid='gameid') }}     Get list of players in a specific game, or add a player to this game
  [GET] {{ url_for('lobbyrest_blueprint.rest_lobby_get_player_info',gameid='gameid', playerid='playerid') }}     Get player specific info in a specific game


TileBagREST
  [GET] {{ url_for('tilebagrest_blueprint.rest_tilebag_get_game_info', gameid='gameid') }}     Get public game info
  [PATCH] {{ url_for('tilebagrest_blueprint.rest_tilebag_trigger_end', gameid='gameid') }}     Trigger the end of the game
  [GET] {{ url_for('tilebagrest_blueprint.rest_tilebag_save_game', gameid='gameid') }}     Get game info in a format that can be reloaded in the future
  [PATCH] {{ url_for('tilebagrest_blueprint.rest_tilebag_placetile', gameid='gameid') }}     Place a tile in a running game
  [PATCH] {{ url_for('tilebagrest_blueprint.rest_tilebag_placehotel', gameid='gameid') }}     Place a hotel in a running game
  [PATCH] {{ url_for('tilebagrest_blueprint.rest_tilebag_stocks', gameid='gameid') }}     Buy or Sell stocks
  [PATCH] {{ url_for('tilebagrest_blueprint.rest_tilebag_money', gameid='gameid') }}     Spend or Receive money (shouldn't have to use this)
